# Agrinho-2025 🌽🎮
Neste jogo seu objetivo é ajudar o personagem Fazendeiro a colher todos os milhos 
🦠 Cuidado! Durante a partida, surgirão diversas bactérias na tela. Evite encostar nelas, pois o personagem possui apenas duas vidas
Como jogar 🎮 
Use as setas W, A, S, D para mover o Fazendeiro.
Colete todos os milhos na tela;
Não toque nas bactérias!

Feito isso espero que aproveite bastante!
